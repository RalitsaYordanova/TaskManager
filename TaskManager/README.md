# Task Manager

A simple console application for managing tasks, built with C# and .NET.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Persist tasks using a JSON file

## Technologies

- C#
- .NET Console Application
- JSON Serialization
