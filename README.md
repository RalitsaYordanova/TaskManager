Task Manager (Console Application)

Simple console-based Task Manager built with C# and .NET.
The application allows users to create, manage, and track tasks directly from the console.

Features

Add new tasks
View all tasks
Mark tasks as completed
Delete tasks with confirmation
Filter tasks (completed / pending)
Search tasks by keyword
Save tasks to a JSON file (data persistence)

Technologies Used

C#
.NET
JSON serialization (System.Text.Json)
LINQ

How It Works

The application stores tasks in a JSON file so that data persists between program runs.
Users interact with the program through a simple console menu.

Example Menu
1 Add task
2 Show tasks
3 Complete task
4 Delete task
5 Show completed tasks
6 Show pending tasks
7 Search tasks
8 Exit
